Description: This node subscribes to the /user_pose topic to retrive the current target position set by the user in the main node. It provides a service called 'user_input' that returns the current x and y coordinates of the target position when requested. This allows other nodes to access the current target position without needing to subscribe to the /usep_pose topic directly.
